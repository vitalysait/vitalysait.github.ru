Домашняя Метеостанция

Скрипт для считывания точных параметров температуры и влажности с датчика BME280 и вывода данных на OLED-дисплей.

C++ (Arduino) 
#include <Wire.h>
#include <Adafruit_BME280.h>

Adafruit_BME280 bme;

void setup() {
  Wire.begin();
  bme.begin(0x76);
}

void loop() {
  float temp = bme.readTemperature();
  delay(2000);
}

RGB Ночник

Плавное управление адресной светодиодной лентой WS2812B для создания неоновых эффектов и Cyberpunk атмосферы.

C++ (FastLED) 
#include <FastLED.h>
#define NUM_LEDS 30
CRGB leds[NUM_LEDS];

void setup() {
  FastLED.addLeds<NEOPIXEL, 6>(leds, NUM_LEDS);
}

void loop() {
  fill_rainbow(leds, NUM_LEDS, 0, 7);
  FastLED.show();
  delay(30);
}

M5StickC Plus2 (ESP32-PICO)

Статус: Идеально под кастомные прошивки (Bruce, Marauder)

Карманный мини-компьютер на базе чипа ESP32 с Wi-Fi и Bluetooth. Версия Plus2 получила увеличенный экран (1.14 дюйма, 135x240 точек) и встроенный аккумулятор на 200 мАч.

Главные фишки:
• IR-передатчик: Позволяет управлять ТВ и кондиционерами удаленно через скрипты.
• Периферия: Встроенный зуммер, микрофон, 6-осевой гироскоп и 3 физические кнопки.
• Автономность: Отлично подходит для запуска JS-скриптов и игровых систем прокачки без внешних SD-карт.

Плата разработки ESP32 NodeMCU

Статус: Народный выбор для умного дома и сетевых задач

Двухъядерный процессор Xtensa с частотой до 240 МГц легко справляется со сложной автоматизацией и веб-серверами.

Технические параметры:
• Связь: Wi-Fi 802.11 b/g/n и Bluetooth v4.2 BLE для беспроводных датчиков или сканирования эфира.
• Память: 4 МБ флеш-памяти вмещают тяжелые скрипты и кастомные веб-интерфейсы.
• Среда: Поддерживает Arduino IDE, MicroPython и JavaScript (Espruino).

Умный паяльник Pinecil V2

Статус: Лучший инструмент на открытой архитектуре RISC-V

Портативный девайс под управлением ОС IronOS, которая дает точнейший контроль температуры и графики на экране.

Преимущества:
• Экстремальный нагрев: До 88W через DC-разъем (24V) или 65W через Type-C от павербанка с PD. Нагрев до 300°C занимает 6-8 секунд.
• Ювелирная точность: Укороченные жала типа Short-Tip гарантируют полный контроль при пайке мелких SMD компонентов.
